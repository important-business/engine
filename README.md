# engine
A game engine of sorts.

## Build Dependencies
 * SDL2
 * SDL2_image
 * Make
 * CMake 3.3

## Uses
 * Google Test
 * Anax - Entity Component System

## Build Instructions
There isn't a whole lot to build at the moment, but...
```
git clone https://github.com/important-business/engine.git engine
cd engine
make
```
